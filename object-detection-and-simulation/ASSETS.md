# Unity asset pack

The simulator's large 3D content is shipped separately to keep the
repository under GitHub's practical size limits. Download the asset
pack and unzip it into `LargeTerrain/Assets/`.

After unzipping, the following four folders should exist directly under
`LargeTerrain/Assets/`:

- `TerrainDemoScene_URP/` -- terrain heightmap, lighting, sky, large
  texture set
- `T90/` -- Soviet T-90 tank model (mesh, textures, materials)
- `Models/` -- military truck, ZPT, civilian vehicle, Akinci UCAV
  meshes and materials
- `Vefects/` -- explosion and impact visual effects

Once those folders are in place, open `LargeTerrain/` in Unity
6000.x. Unity will regenerate `Library/`, `Temp/`, `obj/`, and the
`*.csproj` / `*.sln` files automatically.
